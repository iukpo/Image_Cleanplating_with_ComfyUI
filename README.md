# Image_Cleanplating_with_ComfyUI
This simple ComfyUI workflow demonstrates how to use inpainting to create cleanplates. In this example, we will paint out a traffic controller who is guiding some traffic.

Before running this workflow, be sure to put all images in the ComfyUI/input folder. Main image is from pexels.com.

Generative AI Model: Dreamshaper 8 Inpainting (SD15)
